### Installation:
1. ```composer install```.
2. ```npm install```.
3. ```bower install```.
4. ```typings install```.
5. ```sudo chmod uog+rwx -R storage/```
6. ```sudo chmod uog+rwx -R bootstrap/cache/```
7. Copy ".env.example" to ".env".
8. Enter database credentials to ".env".
9. ```php artisan key:generate```
